# Online Retail Data Analysis using PySpark

## Project Description
This project analyzes online retail data using PySpark in Google Colab.

## Tools Used
- PySpark
- Google Colab

## Steps Performed
- Data loaded from CSV file
- Data cleaned (null values and duplicates removed)
- Negative quantities filtered
- TotalPrice column created
- Data analysis performed

## Analysis Done
- Total Revenue calculation
- Country-wise sales analysis
- Top selling products
- Top customers

## Note
Dataset is not uploaded due to large file size.
